# Todo API

A simple REST API for managing todos. This example demonstrates the core features of OMG (OpenAPI Markdown Grammar).

## Features

- **CRUD operations** - Create, read, update, and delete todos
- **Filtering** - Filter todos by status
- **Pagination** - Limit and offset support

## Authentication

This API uses Bearer token authentication. Include your API key in the `Authorization` header:

```
Authorization: Bearer your-api-key
```
