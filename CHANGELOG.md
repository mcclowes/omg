## [0.1.0] - 2024-12-18

### Added

- Initial release of OAL (OpenAPI Language)
- `@oal/parser` - Markdown-based OAL file parser
- `@oal/compiler` - Compiles OAL to OpenAPI 3.1 specifications
- `@oal/cli` - Command-line interface for OAL compilation
- `oal-vscode` - VS Code extension for syntax highlighting
- Support for API endpoints, parameters, responses, and schemas
- Partial file support for reusable components
- Nested type definitions and recursive schema support
- Comprehensive documentation (DESIGN.md, SYNTAX.md, BEHAVIORS.md, etc.)
